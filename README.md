# CS2 REACTIONS (Jesus Edition) ⚡🔊

**A high-performance, real-time audio companion for Counter-Strike 2.**

CS2 Reactions bridges your in-game moments with immersive audio triggers. Built with a direct-memory audio engine and hardened Game State Integration (GSI), it provides instantaneous audio feedback for every kill, headshot, and death without touching game memory.

---

## 💎 Premium Features

- **Nuclear Audio Engine**: Zero-latency direct memory loading that bypasses Windows security blocks.
- **Tactical HUD**: Integrated real-time GSI sniffer and diagnostics terminal.
- **Global Control**: System-wide hotkey (`Ctrl + Shift + M`) to toggle audio mid-match.
- **Signature Branding**: Custom-designed 'Sound-Bolt' tactical glyph.
- **Portable Profiles**: Export and share your sound configurations via `.cs2vibe` files.

---

## 🚀 Quick Start

1. **Link to CS2**: Open the app and click the link button to automatically install the GSI listener.
2. **Assign Sounds**: Map your favorite `.mp3` or `.wav` files to game events.
3. **Play**: Launch CS2 and listen for the reactions.
